# 服务主管页面

服务主管的规范管理工作台，包含三栏布局：目录、文档编辑器、AI 对话。

## 运行

```bash
npm install
npm run dev
```

访问 http://localhost:3004/supervisor.html

## 技术栈

- React 18 + TypeScript + Vite
- Tailwind CSS
- DashScope Qwen3-max (LLM 对话)
- DashScope CosyVoice V2 (TTS)
