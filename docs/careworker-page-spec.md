# 护理人员页面产品规格

状态：初版
日期：2026-05-17
视觉规范：参照 [global-ui-guidance.md](./global-ui-guidance.md)

## 1. 页面定位

护理人员（护工/上门服务人员）的移动工作台。以 H5 页面形式运行，可嵌入 App 或第三方平台（飞书、微信等）。手机端优先设计。

目标用户：一线护理服务人员（护工），需要查看每日任务排期、了解任务详情、学习服务 SOP。

## 2. 页面结构

```
登录页 → 主页面
              ├── Tab 1: 我的任务（默认）── iOS 风格日历视图
              │                              └── 点击事件 → 任务详情 Drawer
              └── Tab 2: 参考资料 ── SOP 列表
                                      └── 点击条目 → SOP 详情 Drawer
```

## 3. 登录页

### 3.1 功能

- 用户名 + 密码登录表单。
- Demo 模式：默认填入用户名"王建国"、密码自动填充（显示为 `••••••`），用户点击"登录"即可进入。
- 登录后显示顶部 header：用户姓名 + 角色标签（"余杭 · 护理服务人员"）。

### 3.2 UI

- 居中卡片布局，顶部产品标识（"金色年华"）。
- 输入框高度 44px，遵循 `--radius-sm` 圆角。
- 登录按钮使用 Primary Button 样式（`--accent` 底色）。
- 底部显示"智慧养老服务平台"弱文字。

## 4. 主页面框架

### 4.1 顶部 Header

- 高度 56px。
- 左侧：用户 avatar（首字母彩色底 + 姓氏）+ 姓名 + 角色标签。
- 右侧：当前日期显示。

### 4.2 底部 Tab Bar

- 固定底部，两个 Tab：
  - **我的任务**（日历图标）── 默认激活。
  - **参考资料**（书本图标）。
- 图标 + 9-11px 短标签，点击目标高度不小于 44px。
- 激活态：`--accent` 色图标 + 文字；未激活态：`--text-muted`。

## 5. 我的任务（日历视图）

### 5.1 视图切换

右上角 Segmented Control：**日** / **周** / **月**。默认"周"视图。

### 5.2 月视图

- 顶部：年月标题 + 左右箭头切换月份。
- 日历网格：7 列（周日~周六），显示日期数字。
- 有任务的日期下方显示小圆点（颜色对应任务状态）。
- 点击某日 → 下方展开该日任务列表。
- 今天日期使用 `--accent` 圆形底色高亮。

### 5.3 周视图

- 顶部：周范围标题（如"5月12日 - 5月18日"）+ 左右箭头。
- 7 天横向排列，显示 周几 + 日期，今天高亮。
- 下方按天分组显示任务事件列表。
- 每个事件是一个 Rich Row：
  - 左侧色条（状态色）。
  - 服务类型 + 服务对象姓名。
  - 时间段 + 地点。
  - 状态 badge。

### 5.4 日视图

- 顶部：日期标题 + 左右箭头。
- 时间轴布局（8:00 - 18:00），每小时一格。
- 任务以色块显示在对应时间段，高度按时长比例。
- 色块内显示服务类型 + 服务对象姓名。

### 5.5 任务事件颜色

| 状态 | 背景色 | 色条/圆点 |
| --- | --- | --- |
| 已完成 | `rgba(16,185,129,0.1)` 淡绿 | `#10B981` |
| 异常 | `rgba(239,68,68,0.1)` 淡红 | `#EF4444` |
| 待完成（未来） | `rgba(245,158,11,0.1)` 淡橙 | `#F59E0B` |

### 5.6 任务事件摘要格式

```
{服务类型} {服务对象姓名} {时间段} {地点简称}
```

示例："常规探访 陈阿姨 09:00-11:00 翠苑一区"

## 6. 任务详情 Drawer

### 6.1 触发方式

点击日历中的任务事件，从底部弹出 Drawer（手机端）。

### 6.2 Drawer 结构

#### 顶部 Header（带状态色背景条）

- 状态 badge（如"已完成"、"待服务"、"异常"）。
- 服务类型标题（如"助餐"），17px / 700。
- 日期 + 时间（如"5/12 周二 · 09:30-10:30"），13px muted。
- 来源标签（如"周期计划"），tag 样式。
- 底部使用对应状态色的渐变条分隔。
- 右上角关闭按钮（×）。

状态色映射：

| 状态 | Header 背景 | 渐变条色 |
| --- | --- | --- |
| 已完成 | `#F0FDF4` | `#10B981` |
| 异常 | `#FEF2F2` | `#EF4444` |
| 待服务 | `#FFFBEB` | `#F59E0B` |

#### 摘要卡片区（3 列）

| 卡片 | 内容 |
| --- | --- |
| 服务对象 | Avatar（首字母 + 彩色底）+ 姓名 |
| 服务人员 | Avatar（首字母 + 彩色底）+ 姓名 |
| 服务地点 | 定位图标 + 完整地址 |

- 白底 + `--line` 边框 + `--radius-md` 圆角。
- 手机端窄屏时 3 列自动换行为 1 列。

#### 排期详情（3 列网格）

| 字段 | 示例值 |
| --- | --- |
| 服务日期 | 5/12 周二 |
| 时间段 | 09:30-10:30 |
| 服务项目 | 助餐（tag 样式） |
| 来源 | 周期计划 |
| 状态 | 已完成（status badge） |
| 服务人员 | 王丽 |
| 服务对象 | 陈阿姨 |
| 地址 | 上海市杨浦区控江路 1200 号 |
| 备注 | —（无内容时显示破折号） |

- 字段 label 使用 `--accent` 蓝色，12px。
- 字段值使用 `--text`，13px。
- 网格使用 `--line` 分割。
- 手机端窄屏时降级为单列。

### 6.3 Drawer 样式

- 最大高度 86vh，内容区内部滚动。
- `--radius-lg` 上圆角。
- `--shadow-drawer-mobile` 阴影。
- Scrim overlay 点击关闭。

## 7. 参考资料 Tab

### 7.1 SOP 列表

- 按类型分组：**通用规范**、**服务项目规范**。
- 每个 SOP 条目是 Rich Row：
  - 左侧图标（文档图标）。
  - 标题（SOP 名称）。
  - 副标题（版本 + 更新日期）。
  - Trailing disclosure icon（chevron）。
- 点击 → 打开 SOP 详情 Drawer。

### 7.2 SOP 详情 Drawer

- 底部 Drawer，最大高度 86vh。
- Header：SOP 标题 + 关闭按钮。
- 内容区：SOP 全文，Markdown 渲染。
- 支持滚动阅读。

- 区域：余杭

### 8.2 任务数据

覆盖当前周（2026-05-11 ~ 2026-05-17）+ 下周部分，包含：

## 9. 技术实现

- 技术栈：React 18 + TypeScript + Vite + Tailwind CSS。
- 代码位置：`pages/careworker/`。
- 入口：`careworker.html`。
- 纯前端原型，无后端依赖，所有数据使用 mock。
- 手机端优先（375px 基准），适配至桌面端。
